# Курс по Weavy.ai

## Обзор курса

Курс предназначен для изучения платформы Weavy.ai — AI-управляемого инструмента для создания дизайн-воркфлоу, предназначенного для креативных профессионалов.

## Целевая аудитория
- Контент-креаторы
- Маркетологи
- Дизайнеры
- Art Directors
- Команды разработки
- Product Managers

## Структура курса

### Модуль 1: Введение в Weavy.ai
- Что такое Weavy.ai и Artistic Intelligence
- Обзор платформы и возможностей
- Регистрация и настройка аккаунта
- Тарифные планы: Collective vs Enterprise
- Интеграция с Figma

### Модуль 2: Основы интерфейса
- Node-based подход
- Навигация по рабочему пространству
- Создание первого workflow
- Режимы работы: Workflow vs App Mode
- Dashboard и управление проектами

### Модуль 3: AI-модели изображений
- GPT Image 1
- Stable Diffusion 3.5
- Imagen 3
- Recraft V3
- Flux Pro 1.1 Ultra
- Ideogram V3
- Сравнение моделей и выбор подходящей

### Модуль 4: AI-модели видео
- Runway Gen-4
- Veo 3
- Kling
- Minimax Video
- Luma Ray 2
- Создание видеоконтента

### Модуль 5: Инструменты редактирования
- Базовое редактирование (Default, Crop, Blur)
- Маскирование и селекция (Mask, Inpaint, Outpaint)
- Работа с цветом (Invert, Channels)
- Улучшение качества (Upscale)
- Освещение (Relight)
- Глубина и 3D (Z Depth)

### Модуль 6: Композитинг и слои
- Работа со слоями
- Типографика
- Режимы наложения
- Создание сложных композиций
- Best practices композитинга

### Модуль 7: Продвинутые workflow
- Multiple Models workflow
- ControlNet и Structure Reference
- Wan LoRA и трансформации
- Camera Angle Control
- Создание собственных workflow

### Модуль 8: Практические проекты
- Создание маркетинговых материалов
- Редактирование продуктовой фотографии
- Создание концепт-артов
- Видеопроизводство
- Автоматизация workflow

### Модуль 9: Командная работа
- App Mode для команд
- Совместное использование workflow
- Управление доступом
- Интеграция с Figma для команд

### Модуль 10: Best Practices и оптимизация
- Оптимизация workflow
- Экономия ресурсов
- Организация проектов
- Безопасность и SOC 2

## План разработки

1. Создать структуру проекта
2. Разработать учебные материалы для каждого модуля
3. Создать практические примеры кода
4. Подготовить демонстрации и скриншоты
5. Написать тесты для проверки знаний
6. Провести ревью и обновление

## Технические требования

- Браузер: Chrome, Firefox, Safari (последние версии)
- Интернет-соединение: стабильное, минимум 10 Мбит/с
- Аккаунт Weavy.ai (рекомендуется Collective для полного доступа)
